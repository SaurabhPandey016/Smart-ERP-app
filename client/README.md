> This is the polished frontend layer of SmartERP — a modern, responsive, business-focused interface built to make ERP workflows feel fast, intuitive, and professional.

## What the client delivers

- A clean dashboard experience for daily operational views
- Company-aware navigation and business module flows
- Interactive forms and tables for ledgers, vouchers, suppliers, customers, and stock
- Protected routes and state-driven application behavior
- A premium UI foundation designed to feel like a serious internal tool, not a basic demo

## Core frontend capabilities

- Authentication screens and protected access
- Company management and business context switching
- Inventory, vouchers, reports, and ledger pages
- Responsive layout for desktop and tablet usage
- Smooth state management with Zustand and React hooks

## Tech stack

| Area | Stack |
|---|---|
| Framework | Next.js 16 + React 19 |
| Styling | Tailwind CSS |
| State | Zustand |
| Forms | React Hook Form + Zod |
| UI Components | Radix UI, Lucide Icons |
| Data Fetching | Axios |

## Project structure

```text
client/
├── app/            # App router pages and routes
├── components/     # Reusable UI building blocks
├── lib/            # API, state, and utility layers
├── public/         # Static assets
└── package.json    # Frontend dependencies and scripts
```

## Run locally

```bash
cd client
npm install
npm run dev
```

Open http://localhost:3000 to view the app.

## Design philosophy

This frontend emphasizes clarity, speed, and business usability. It is built to reflect a professional ERP experience where operations feel structured, modern, and dependable.

## Highlights

- Modular page architecture
- Reusable component system
- Strong separation between UI and API concerns
- A polished experience ready for portfolio presentation or team review
